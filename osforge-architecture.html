🎯 Triage de Complexidade

Classificação automática que determina o pipeline de execução

QUICK

1-3 arquivos, zero ambiguidade

Fix, ajuste, refactor pontual, campo, cor, validação, lint, README simples

spec → implement → review

STANDARD

Feature multi-arquivo, domínio conhecido

CRUD, API, endpoint, mobile screen, game feature, Docker, profiling

spec → arch → stories → loop → review

COMPLEX

Sistema novo, requisitos ambíguos, multi-integração

SaaS, marketplace, jogo completo, app mobile do zero, microsserviços, multi-tenant

PRD → arch → epics → gate → sprints
