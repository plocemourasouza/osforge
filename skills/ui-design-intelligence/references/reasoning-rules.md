# Reasoning Rules — UI Design Intelligence

161 regras de raciocínio por tipo de produto/indústria (motor de design system do v2.0). Mapeia produto → pattern + estilo + cor + tipografia + efeitos + anti-patterns.

### SaaS (General)
**Pattern:** Hero + Features + CTA  
**Estilo:** Glassmorphism + Flat Design  
**Cor:** Trust blue + Accent contrast  
**Tipografia:** Professional + Hierarchy  
**Efeitos:** Subtle hover (200-250ms) + Smooth transitions  
**Anti-patterns:** Excessive animation + Dark mode by default
